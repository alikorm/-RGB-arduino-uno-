# -RGB-arduino-uno-
في عالم يملأه بالاكترونيات وروبوتيكس يتواجد Arduino وArduino ليس مجرد لوح هذه فكره تم انشائها تقريبا فتره في فترات قبل 2010 وفي 2010 حدثت ثوره التي ستغير عالم روبوتيكس للابد وهي arduino uno ليس فقط انه احدث قوره بل هو كان بسيط جدا وساعد كثير على تعرف على اردوينو ودخل مجال من اول مقدمه لحدمافهم ماهي مكثفاته وماهي مداخل Analoge بوكني تعلن كود منCEO
